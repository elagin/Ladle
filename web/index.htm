Рыжий лис

Готовить с нами еще удобнее.

Возможности:

  • Хранение рецепта с пошаговым описанием.
  • Загрузка фотографий для рецепта и каждого шага.
  • Поделиться рецептом через email с другим пользователем приложения см. "Поделиться файлом" и "Загрузить рецепт".
  • Поделиться текстом рецепта через email, WhatsUp, Vk и т.д.
  • Пошаговое приготовление блюда.

Планы:

  • Использование таймера в шагах приготовления.
  • Формирование списка покупок.

Чаво:

  • Как поделиться рецептом?
  • В верхней панели окна просмотра рецепта есть кнопка "поделиться" в которой есть два пункта:
    • Поделиться текстом можно использовать для тех у кого не установлено наше приложение. Сформируется обычный текст, который можно будет отправить любым удобным способом: По СМС, WhatsUp, Viber, VK, email и т.д.
    • Поделиться файлом можно использовать для других пользователей нашего приложения. Сформируется специальный файл, который можно будет отправить через WhatsUp, Viber, VK, email. Получателю нужно будет его сохранить на свое устройство и в главном экране воспользоваться пунктом меню "Загрузить рецепт".
